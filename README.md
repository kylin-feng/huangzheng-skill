## 效果示例

### 问：我们要进入一个已经有大玩家的市场，还有机会吗？

```
黄峥    ❯ 先不要问「有没有机会」，先问「这个市场里有哪些大家都接受的假设」。

          大玩家存在，说明他们解决了某些问题。
          但他们在解决问题的过程中，一定接受了一批假设——
          有些假设是对的，有些假设只是「大家都这么做」。

          找到那个「大家都接受但其实是错的」假设，就是你的入口。

          拼多多进入电商市场的时候，
          有一个被所有人接受的假设：「电商需要搜索框，用户知道自己要买什么」。
          我们发现：很多用户其实不知道自己要买什么，他们在逛。
          这个假设是错的。

          你的市场里，有哪个假设可能是错的？
```

### 问：我在犹豫要不要接受一笔投资，但条件要求我们快速增长，你怎么看？

```
黄峥    ❯ 这取决于你的增长是「创造了新价值」还是「加速了价值分配」。

          快速增长本身不是坏事。
          但如果快速增长的方式是：烧钱抢用户、补贴挤对手、
          然后等对手死了再涨价——
          这是价值分配，不是价值创造。
          这种增长，停止补贴就会停止。

          真正的增长应该来自：
          你做了一件之前没人做的事，用户因为这件事选择你。
          这种增长，竞争对手很难复制，因为他们没有你的供应链/技术/用户理解。

          问自己：如果明天停止所有补贴，用户还会选我吗？
          如果答案是否，这笔钱接了，是在买时间，不是在建护城河。
```

## 蒸馏了什么

### 5 个心智模型

| 模型 | 一句话 | 来源 |
|------|--------|------|
| **本分哲学** | 创造新价值，而非分配已有价值 | 段永平传承，农产品直供链路建设 |
| **消费者剩余** | 把便宜留给用户，用户会替你传播 | 百亿补贴，农产品直供定价 |
| **反常识思维** | 找行业「大家接受但其实是错的」假设 | 无搜索框社交拼团，打破农产品电商假设 |
| **农村包围城市** | 从主流平台忽视的用户群切入 | 下沉市场起点，再向上渗透 |
| **少做事，做对的事** | 聚焦是最被低估的战略能力 | 主动卸任，专注基础研究 |
